# Filtered Lie algebroids and parabolic geometry (version 1)

Anthony D. Blaom

My notes on [parabolic
geometry](https://en.wikipedia.org/wiki/Parabolic_geometry_(differential_geometry)), from
the **Lie algebroid/groupoid** point of view, are posted here. As these are developing over an
extended period of time, I am making them publicly available, before posting a
"final" version on the arXiv.

The notes are subject to change at any time. 

While this is a work in progress, versions posted here will be "complete" in the
sense that each topic presented is complete. Future versions may add new material,
examples, and generalizations.
